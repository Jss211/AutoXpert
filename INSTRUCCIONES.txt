# 📋 INSTRUCCIONES PARA USAR EL SISTEMA DE EMAIL

## ¿QUÉ SE IMPLEMENTÓ?

✅ **Formulario de contacto mejorado** con:
  - Selector de país con códigos automáticos (+51 Perú, +52 México, +57 Colombia, etc.)
  - Validación en tiempo real
  - Notificación visual profesional al enviar (SIN LOADER)

✅ **Backend con Node.js/Express** que:
  - Recibe los datos del formulario
  - Envía email profesional a tu Gmail
  - Incluye HTML hermoso y bien diseñado

✅ **Email profesional** que incluye:
  - Logo de AutoXpert
  - Datos del cliente (nombre, email, teléfono, país, interés)
  - Mensaje completo del cliente
  - Botón para responder rápidamente
  - Información de contacto

## PASOS PARA PROBAR (MUY IMPORTANTE!)

### 1️⃣ GENERAR CONTRASEÑA DE GMAIL (OBLIGATORIO)

No puedes usar tu contraseña normal de Gmail. Debes generar una contraseña de aplicación:

1. Ve a https://myaccount.google.com/
2. Click en "Seguridad" (izquierda)
3. Busca "Contraseñas de aplicación"
4. Selecciona "Correo" y "Windows"
5. Google te dará una contraseña de 16 caracteres
6. CÓPIALA

### 2️⃣ CONFIGURAR EL ARCHIVO .env

Abre el archivo `.env` en la carpeta del proyecto y reemplaza:

```
EMAIL_USER=jordanpmrojasbazan@gmail.com
EMAIL_PASSWORD=aqui_pega_los_16_caracteres
PORT=3000
```

Ejemplo:
```
EMAIL_USER=jordanpmrojasbazan@gmail.com
EMAIL_PASSWORD=xyza bcde fghi jklm
PORT=3000
```

### 3️⃣ INICIAR EL SERVIDOR

Abre PowerShell o Terminal en la carpeta del proyecto y ejecuta:

```
npm start
```

Deberías ver:
```
✅ Servidor AutoXpert corriendo en http://localhost:3000
```

### 4️⃣ PROBAR EL FORMULARIO

1. Abre el proyecto en el navegador: http://localhost:5500 (o tu puerto local)
2. Scroll hasta "Contáctanos"
3. Llena el formulario:
   - Nombre: Tu nombre
   - Email: Tu email
   - País: Selecciona uno (ej: Perú)
   - Teléfono: 987654321
   - Interés: Elige uno
   - Mensaje: Escribe algo
4. Click en "Enviar mensaje"
5. Deberías ver una **notificación verde** diciendo "¡Mensaje Enviado!"
6. Revisa tu Gmail - el email debería haber llegado 📧

## ¿QUÉ PASA CUANDO ENVÍAS?

## SOLUCIONAR PROBLEMAS

### ❌ "Error de conexión"
- ¿El servidor está corriendo? Verifica PowerShell
- Ejecuta: `npm start`
- Espera a ver: `✅ Servidor AutoXpert corriendo en http://localhost:3000`

### ❌ "Error al enviar el mensaje"
- Verifica que la contraseña en .env es correcta
- Revisa tu Gmail: puede haber alerta de seguridad
- Intenta de nuevo

### ❌ El email no llega
- Revisa SPAM en Gmail
- Verifica que EMAIL_USER es correcto en .env
- Asegúrate de usar la contraseña de aplicación, no la normal

### ❌ El selector de país no funciona
- Abre Developer Tools (F12)
- Verifica si hay errores en la consola
- El servidor debe estar corriendo
